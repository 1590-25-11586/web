Pasos para Crear el Sitio

  1. Se creó el archivo index.html utilizando la etiqueta ;frameset; para dividir la página en secciones.
  2. Se diseñó un archivo menu.html que funciona como menú de navegación, utilizando enlaces apuntando a cada sección del contenido.
  3. Se crearon las páginas de contenido correspondientes: hoja de vida, tabla, galería, video, juego y manual, cada una con su propia estructura HTML.
  4. Se utilizó HTML para estructurar el contenido en cada página, incluyendo encabezados, párrafos, listas, tablas, formularios y multimedia según el tema.
  5. Se aplicaron estilos básicos con CSS directamente en cada archivo HTML o en una hoja de estilo externa para mejorar la presentación visual.
  6. Se integró JavaScript en la página del juego para hacerlo interactivo, añadiendo funciones para responder a eventos del usuario.
  7. Se configuró el atributo target en los enlaces del menú para que las páginas se abrieran dentro del frame correspondiente.
  8. Se probaron todos los enlaces y funcionalidades para asegurar que cada sección cargara correctamente en su lugar dentro del ;frameset;.
  9. Se revisó la compatibilidad en distintos navegadores para asegurar que la página se visualizara correctamente.
  10. Finalmente, se organizaron los archivos en carpetas (por ejemplo, img para imágenes y css para estilos) para mantener una estructura limpia del proyecto.
